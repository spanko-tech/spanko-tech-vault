This repository contains systems from my personal vault that I use daily.

Full discretion: All of the code was generated by GitHub Copilot - Claude Opus 4.7. I am not proficient in JS and React enough to really check the code. Obsidian is one of the few things I do not mind fully vibe coding.

I had it extract all the reusable parts into a library of sorts, but if you see something horrendous that is probably why.

## Tech/Plugins Used
All of the systems are utilizing [Datacore](https://blacksmithgu.github.io/datacore/) (the replacement for Dataview from the same [author](https://github.com/blacksmithgu)).
Very small portions use Templater to run scraper scripts that I had from pervious Dataview version of the vault.

Other plugins include Excalidraw and Folder Notes - both can be removed (however Excalidraw removal needs changes in Cogito)
